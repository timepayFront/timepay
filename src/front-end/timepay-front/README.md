# 타임페이 커뮤니티 사용자 사이트

## 프로젝트 스크립트

### 로컬 실행하기

```shell
 yarn install
 yarn start
```

### 빌드

```shell
 yarn build
```

## 프로젝트 구조

┣ src

┃ ┣ assets    
┃ ┃ ┗ images

┃ ┣ components  
┃ ┣ hooks // custom hooks  
┃ ┣ pages  
┃ ┣ states // recoil states
